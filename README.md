# Exploratory-Manual-Testing-Bugs-Found-On-Czech-Jewelry-Website
This project serves as a demonstration of my work, thinking and knowledge to track down bugs independently of any guidance.
It shows my hands-on approach to exploratory testing and user-first thinking. By using software QA methodology and DevTools, some common bugs degrading the overall user experience were inspected, identified and documented. Each screenshot represents one bug found, together with my commentaries and DevTools displayed in it. Many more projects are to come!

Note: To protect the company's privacy, manual test cases aren't being published here due to containing sensitive information that could be misused (links leading to a real website I anonymized). However, complete test documentation will be available upon a request for companies interested in reviewing my methodology. The names of the screenshot pictures also serve as a brief explanation of what's wrong with the website design/functioning.
